What is an Operating System?

An operating system is the main program that runs on your computer. It's like a manager that helps all your apps work together!

Simple Explanation: Think of the operating system as a traffic controller - it makes sure all programs take turns using the computer!

Common Operating Systems

  • Windows: Used on most PCs
  • macOS: Used on Apple computers
  • Linux: Popular for servers and developers

Operating System Fundamentals

Operating systems manage hardware resources, provide services for applications, and create a user interface for interaction with the computer.

Core OS Functions

  • Process Management: Running multiple programs
  • Memory Management: Allocating RAM to programs
  • File Systems: Organizing data on disk
  • I/O Management: Handling devices

Example: Process Information

import os
import psutil

print(f"Process ID: {os.getpid()}")
print(f"CPU Usage: {psutil.cpu_percent()}%")
print(f"Memory: {psutil.virtual_memory().percent}% used")

Advanced Operating Systems

Understanding kernel internals, scheduling algorithms, memory management techniques, and system calls is crucial for systems programming.

OS Internals

  • Scheduling Algorithms: Round-robin, Priority, Multilevel Queue
  • Virtual Memory: Paging, Segmentation, TLB
  • Synchronization: Mutexes, Semaphores, Monitors
  • Deadlock: Prevention, Avoidance, Detection

Example: Threading and Synchronization

import threading

counter = 0
lock = threading.Lock()

def increment():
    global counter
    for _ in range(100000):
        with lock:
            counter += 1

threads = [threading.Thread(target=increment) for _ in range(4)]
for t in threads: t.start()
for t in threads: t.join()
print(f"Counter: {counter}")
