What is Programming?

Programming is like giving instructions to a computer. Just like you follow a recipe to make a cake, a computer follows your instructions to do tasks!

Simple Explanation: Think of a computer as a very fast helper that does exactly what you tell it to do.

Your First Code

Here's a simple example that tells the computer to say "Hello":

print("Hello, World!")

When you run this code, the computer will display: Hello, World!

Introduction to Programming

Programming is the process of creating instructions that a computer can execute. These instructions are written in programming languages like Python, Java, or JavaScript.

Key Concepts

  • Variables: Containers that store data
  • Data Types: Different kinds of data (numbers, text, etc.)
  • Operators: Symbols that perform operations
  • Control Flow: Making decisions in code

Example: Variables and Output

name = "Student"
age = 20
print(f"Hello, {name}! You are {age} years old.")

Programming Fundamentals

Programming involves designing algorithms, implementing data structures, and understanding computational complexity to solve problems efficiently.

Core Programming Paradigms

  • Imperative: Step-by-step instructions
  • Declarative: Describing what to achieve
  • Object-Oriented: Organizing code into objects
  • Functional: Using pure functions

Example: Function with Error Handling

def calculate_average(numbers: list) -> float:
    """Calculate the average of a list of numbers."""
    if not numbers:
        raise ValueError("Cannot calculate average of empty list")
    return sum(numbers) / len(numbers)

try:
    result = calculate_average([85, 90, 78, 92])
    print(f"Average score: {result:.2f}")
except ValueError as e:
    print(f"Error: {e}")
