What are Data Structures?

Data structures are ways to organize information on a computer, like different types of containers for storing things!

Simple Explanation: Think of a data structure like a backpack with different pockets - each pocket helps you find things quickly!

Lists - Like a Shopping List

A list keeps items in order, one after another:

fruits = ["apple", "banana", "orange"]
print(fruits[0])  # Shows: apple

Introduction to Data Structures

Data structures are specialized formats for organizing, processing, and storing data. The choice of data structure affects the efficiency of your programs.

Common Data Structures

  • Arrays/Lists: Sequential collection of elements
  • Dictionaries/Maps: Key-value pairs
  • Sets: Unique elements only
  • Stacks: Last-In-First-Out (LIFO)
  • Queues: First-In-First-Out (FIFO)

Example: Using Dictionaries

student = {
    "name": "Alice",
    "age": 20,
    "grades": [85, 90, 78]
}
print(f"{student['name']} is {student['age']} years old")

Example: Implementing a Stack

class Stack:
    def __init__(self):
        self._items = []
    
    def push(self, item):
        self._items.append(item)
    
    def pop(self):
        if not self.is_empty():
            return self._items.pop()
        raise IndexError("Stack is empty")
    
    def is_empty(self):
        return len(self._items) == 0
